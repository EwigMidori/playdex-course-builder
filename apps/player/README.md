# Playdex Player

Learner-facing React app for the generated course platform.

- `viewer` is the creator/debug surface.
- `player` is the customer learning product.

The player reads directly from `research/pipeline/course-index.json` and the generated guided story, question bank, and textbook assets.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
