## 为什么这一课重要 {#why-this-lesson-matters}

在投入真金白银之前，如何知道你的交易策略是否有效？这一课将带你进入量化交易的核心环节——**回测**。你将学习如何用历史数据模拟交易策略，验证其有效性，并掌握从数据清洗到策略实现、从Excel到Python的完整回测框架。这是从理论走向实践的关键一步。

<KeyPoint>
回测（Backtesting）是用历史数据模拟交易策略，验证其有效性的过程。它是量化交易的生命线。
</KeyPoint>

## 回测基础：定义、流程与数据清洗 {#backtesting-foundation}

### 什么是回测？

<Definition title="回测 (Backtesting)">
用历史数据模拟交易策略，验证其在过去表现如何的过程。其核心目的是在实盘交易前评估策略的有效性，避免盲目投资。
</Definition>

回测就像电影上映前的试映会——通过小范围观众的反馈来预测电影是否受欢迎，而不是直接大规模上映。

<KeyPoint>
回测的四大关键步骤：
1. 数据收集 (Data Collection)
2. 数据清洗 (Data Cleaning)
3. 策略实施 (Strategy Implementation)
4. 绩效评估 (Performance Evaluation)
</KeyPoint>

### 数据清洗：让数据可信

原始数据中常常包含各种问题，直接使用会导致回测结果失真。这个过程称为**数据清洗**。

<Definition title="数据清洗 (Data Cleaning)">
识别并修正数据中的错误、缺失、重复或不一致，以提高数据质量，为后续分析打好基础。
</Definition>

#### 常见的数据问题

1.  **缺失值 (Missing Values)**：某一天的价格数据为空。
    -   **处理方法**：从其他来源补全、用相邻数据填充（如取平均值）、或直接删除该行。
2.  **重复记录 (Duplicated Records)**：同一天出现两条完全一样的数据。
    -   **处理方法**：保留一条即可。
3.  **逻辑错误 (Incorrect Logics)**：数据违反了基本逻辑，例如收盘价高于当日最高价。
    -   **处理方法**：从其他来源核对、删除该行、或将错误值修正为边界值（如将收盘价修正为最高价）。

<Example title="逻辑错误识别">
在Excel中，可以使用公式 `=AND(Close<=High, Close>=Low, Open<=High, Open>=Low)` 来快速检查一行数据是否有效。
</Example>

## K线图与常见形态 {#candlestick-patterns}

### K线图基础

<Definition title="K线图 (Candlestick Chart)">
一种展示资产价格波动的金融图表，它将一天（或任意时间段）的价格变化浓缩成一根“蜡烛”，直观地显示开盘价、收盘价、最高价和最低价（OHLC）。
</Definition>

<Figure src="images/a6fccb6cb2b229b6494f629e95a068ae057b8742065c1b4687634aa1d4a2283d.jpg" alt="K线图组成部分示意图，标注了实体、上影线、下影线、开盘价、收盘价、最高价和最低价">
K线图的组成部分
</Figure>

-   **实体 (Body)**：开盘价与收盘价之间的区域。
-   **影线 (Wicks)**：从实体上下延伸的线条，代表最高价和最低价。
-   **阳线 (Bullish Candle)**：收盘价 > 开盘价，通常为绿色或白色。
-   **阴线 (Bearish Candle)**：收盘价 < 开盘价，通常为红色或黑色。

## 移动平均线交叉策略 {#ma-cross-strategy}

### 移动平均线 (Moving Average)

<Definition title="移动平均线 (Moving Average, MA)">
对一定时期内的价格取平均值，用于平滑短期波动，显示长期趋势。
</Definition>

**简单移动平均线 (Simple Moving Average, SMA)** 是最基础的形式：

<Formula latex="\mathrm{MA}(n) = \frac{\sum_{t=1}^{n} P_t}{n}">
N日简单移动平均线等于过去N天的收盘价之和除以N。
</Formula>

<Example title="SMA计算示例">
过去5天的收盘价分别为 [10, 12, 14, 16, 18]，则5日SMA = (10+12+14+16+18) / 5 = 14。
</Example>

<QuestionFamily familyId="qf_flash_ma_def" />
<QuestionRef id="q_quiz_ma_calc_v1" />

### MA交叉策略

MA交叉策略使用两条不同周期的移动平均线：一条**快线**（短期，如7日MA）和一条**慢线**（长期，如14日MA）。

-   **黄金交叉 (Golden Cross)**：快线上穿慢线，产生**买入**信号。
-   **死亡交叉 (Death Cross)**：快线下穿慢线，产生**卖出**信号。

## 用Excel实现MA交叉策略 {#excel-implementation}

在Excel中实现MA交叉策略，可以让你直观地理解回测的每一个步骤。以下是核心流程：

1.  **计算移动平均线**：使用 `AVERAGE` 函数计算MA(7)和MA(14)。
    -   例如，`=AVERAGE(F2:F8)` 计算第2到第8行的收盘价平均值，即7日SMA。
2.  **识别交叉信号**：使用 `IF` 和 `AND` 函数判断黄金交叉和死亡交叉。
    -   买入信号 (1)：`=IF(AND(H16>I16, H15<I15), 1, ...)`
    -   卖出信号 (-1)：`=IF(AND(H16<I16, H15>I15), -1, 0)`
3.  **计算每日持仓**：根据信号和前一日的持仓更新持仓。
    -   例如，`=K2+IF(AND(K2=0, J3=-1), 0, J3)`
4.  **计算每日盈亏**：`= 持仓 * (今日收盘价 - 昨日收盘价)`
5.  **计算累计盈亏**：累加每日盈亏。

### Backtesting.py

<Definition title="Backtesting.py">
一个轻量级的Python回测框架。你只需定义一个策略类，在 `init` 方法中初始化指标，在 `next` 方法中编写交易逻辑，框架会自动执行回测并生成绩效报告和图表。
</Definition>

```python
from backtesting import Backtest, Strategy
from backtesting.lib import crossover
from backtesting.test import SMA

class SmaCross(Strategy):
    def init(self):
        price = self.data.Close
        self.ma1 = self.I(SMA, price, 10)
        self.ma2 = self.I(SMA, price, 20)

    def next(self):
        if crossover(self.ma1, self.ma2):
            self.buy()
        elif crossover(self.ma2, self.ma1):
            self.sell()

bt = Backtest(GOOG, SmaCross, cash=10000, commission=0.002)
stats = bt.run()
bt.plot()
```

## 回测的陷阱与假设 {#pitfalls-assumptions}

回测结果看起来很漂亮？先别急着高兴。回测中有很多陷阱，一不小心就会掉进去。

### 常见陷阱

1.  **过度拟合 (Overfitting)**：策略被调整得过于贴合历史数据，导致在新数据上表现不佳。
    -   **避免方法**：使用样本外数据验证，保持策略简洁。
2.  **前视偏差 (Look-ahead Bias)**：在回测中不小心使用了当时无法获得的未来信息。
    -   **例子**：用今天的收盘价来决定昨天的交易。
    -   **避免方法**：确保所有用于决策的数据在决策时点都是已知的。
3.  **幸存者偏差 (Survivorship Bias)**：只测试了现在还活着的股票，忽略了那些已经退市的失败者。
    -   **后果**：高估策略表现。
    -   **避免方法**：使用包含所有历史资产（包括已退市）的数据集。

### 不现实的假设

回测中常做的假设，如“无交易成本”、“可以按收盘价成交”、“允许卖空”，在真实市场中往往不成立。交易成本、滑点、流动性限制都会影响最终收益。

<KeyPoint>
回测结果好，不代表实盘也能赚钱。关键是要用现实的假设，并在样本外数据上验证。
</KeyPoint>

## RSI策略练习 {#rsi-strategy}

### 相对强弱指标 (RSI)

<Definition title="相对强弱指标 (Relative Strength Index, RSI)">
一个动量指标，用于衡量价格变动的速度和幅度，取值范围在0到100之间。通常用于识别市场的超买或超卖状态。
</Definition>

-   **超买 (Overbought)**：RSI > 70，可能面临回调。
-   **超卖 (Oversold)**：RSI < 30，可能迎来反弹。

### RSI策略规则

一个简单的RSI策略如下：

-   当 `RSI(14) < 30` 时，**买入**。
-   当 `RSI(14) > 70` 时，**卖出**。
